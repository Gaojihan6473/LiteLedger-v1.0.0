# LiteLedger 轻账

个人记账移动优先 Web 应用。

## 功能

- 记录支出和收入
- 查看交易历史
- 分析消费统计图表
- 追踪储蓄目标
- 日历视图查看记录

## 开发

```bash
npm install        # 安装依赖
npm run dev       # 启动开发服务器（端口 3000）
npm run build     # 生产环境构建
npm run preview   # 预览生产构建
npm run lint      # TypeScript 类型检查
```

## 技术栈

- React 19 + Vite 6
- TypeScript
- Zustand + localStorage
- Tailwind CSS v4
- recharts
- lucide-react
